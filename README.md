# Meta-Ad-Performance-Analysis
This dataset represents Meta Ads Performance Data, covering campaigns, ads, user  demographics, and ad interaction events. It is modelled after how Facebook/Instagram  (Meta) ad platforms capture data.  The purpose of this dataset is to analyse advertising performance, optimize targeting, and  measure ROI (Return on Investment) through KPIs
